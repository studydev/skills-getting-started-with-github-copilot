# GitHub Copilot 시작하기

_1시간 이내에 GitHub Copilot 사용을 시작해 보세요._

## 환영합니다

- **대상**: 코드 작업 흐름을 가속화하고자 하는 모든 수준의 개발자
- **학습 내용**: Copilot과 상호작용하여 코드를 설명, 작성, 계획, 개발하는 다양한 방법
- **실습 내용**: Copilot을 활용하여 Mergington 고등학교의 과외 활동 웹사이트를 업데이트합니다.
- **사전 요구 사항**:
  - Skills 실습: [GitHub 소개](https://github.com/skills-kr/introduction-to-github)
  - [VS Code](https://code.visualstudio.com/) 사용 경험
  - 기본 코딩 원리 이해
- **소요 시간**: 이 실습은 1시간 이내에 완료할 수 있습니다.

이 실습에서는 다음을 수행합니다:

1. 사전 구성된 Codespace를 사용하여 브라우저에서 VS Code를 실행합니다.
1. GitHub Copilot으로 개발하고 계획하는 다양한 상호작용 옵션을 배웁니다.
1. Copilot을 사용하여 Pull Request를 요약하고 리뷰합니다.

### 실습을 시작하는 방법

실습을 여러분의 계정으로 복사한 후, Octocat(Mona)이 첫 번째 레슨을 준비할 때까지 **약 20초** 기다린 다음 **페이지를 새로고침**하세요.
